    拿到数据之后首先是对数据做分析(特征工程),该notebook分析的方式是：
    (1) 首先对所有数据做简单分析：数据前几行head,数据info和数据describe,查看部分数据之间的关系
    (2) 然后逐个对各个特征做处理和分析，如:
            ①优惠比率Discount_rate 从原来的N:M形式获取比率,需够满的金额(符合常理),优惠金额
            单独拆分出须够满的金额和优惠金额作为新的特征符合实际消费心理。
            ②时间：时间分购物券领取时间和使用时间，领取时间和消费时间的延时，领取时间是周末还是工作日
                   不同时间购物券的领取数和消费数
